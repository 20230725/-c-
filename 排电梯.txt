小乐乐学校教学楼的电梯前排了很多人，他的前面有n个人在等电梯。电梯每次可以乘坐12人，每次上下需要的时间为4分钟（上需要2分钟，下需要2分钟）。请帮助小乐乐计算还需要多少分钟才能乘电梯到达楼上。（假设最初电梯在1层）

输入描述：
输入包含一个整数n (0 ≤ n ≤ 109)

输出描述：
输出一个整数，即小乐乐到达楼上需要的时间。
示例1
输入：
1
复制
输出：
2
复制
示例2
输入：
12
复制
输出：
6

#include <stdio.h>
int main() {
    int n,i=0;
    scanf("%d",&n);
    while(n-11>0){
        i+=4;
        n-=12;
    }
    i+=2;
    printf("%d",i);
    return 0;
}
